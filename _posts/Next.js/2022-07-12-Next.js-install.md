# Antd를 Next.js 설치하기

1. 일단 Antd에 yarn 혹은 npm을 이용해서 설치한다. 아이콘도 함께 이용한다면 아이콘도 함께 설치한다.

```
yarn add antd @ant-design/icons
npm install antd @ant-design/icons --save
```

2. `antd/dist/antd.css` 를 불러온다.

```css
/* global.css */
@import "antd/dist/antd.css";
```
