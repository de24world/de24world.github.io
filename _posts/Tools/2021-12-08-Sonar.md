# SonarQube란?

SonarQube는 소스 코드를 분석한 후에 버그(Bugs), Code Smells, 중복(Duplication) 등을 분석하는 프로젝트의 품질을 관리할 수 있는 정적 분석 도구이다. 클라우드 서비스를 제공해주는 SonarCloud가 있고, eslint나 stylelint처럼 IDE Extension이나 Plugin 처럼 사용할 수 있는 Sonarlint가 있다.
