# GTM(Google Tag Manager, 구글 태그 매니저)

## GTM vs gtag.js

| 구분 | Google Tag Manager                         | gtag.js                                   |
| ---- | ------------------------------------------ | ----------------------------------------- |
| 정의 | 웹/웹 추적코드 관리시스템                  | 웹페이지에 직접 추가하는 자바스크립트 F/W |
| 특징 | 웹 인터페이스에서 웹앱 태그 수정 및 배포용 | 직접 설치/운영, 자바스크립트 작업이 중요  |
| 사용 | 중대형 웹사이트                            | 중소형                                    |
| 공통 | 구글 범용 제품에 적용 관리 가능            |
