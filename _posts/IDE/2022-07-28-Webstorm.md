# 기타 추천

- [React,Redux Snippet](https://github.com/ifahrentholz/webstorm-snippets)

1. `live_template.jar` 파일을 다운로드한다.
2. 웹스톰을 연다 (혹은 다른 IDEA IDE)
3. File > Manage IDE Settings > Import Settings 클릭해서 `live_template.jar` 파일을 열어준다.
