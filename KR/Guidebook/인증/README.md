
[인증, 인가] 
======================

# 인증(Authenfication)
* '인증'은 로그인, 회원가입 부분에서 사용한다. 서버에 '인증'은 모든 api 요청에 대해 `사용자를 확인`하는 작업이다. 

# 인가(Authorization)
* `권한 부여`를 의미하며, 클리아이언트가 하고자 하는 작업이 해당 클라이언트에게 허가된 작업인지 확인하는 절차. 즐, 해당 리소스에 접근할 수 있는 권한이 있는지 확인하는 절차.
